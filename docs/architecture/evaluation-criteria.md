# Evaluation Criteria

Every resource is evaluated using the following criteria.

| Criterion | Weight |
|-----------|-------:|
| Practical Value | 25% |
| Compatibility | 20% |
| Community | 15% |
| Documentation | 15% |
| Maintainability | 10% |
| Learning Value | 10% |
| Future Potential | 5% |

Final score determines recommendation level.
