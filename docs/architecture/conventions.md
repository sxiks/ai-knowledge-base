# Naming Conventions

## Language

English.

---

## File Naming

Use kebab-case.

Example:

recommended-agents.md

---

## Directory Naming

Use lowercase.

---

## Markdown

Use ATX headings.

---

## Metadata

Every resource starts with YAML Front Matter.

---
